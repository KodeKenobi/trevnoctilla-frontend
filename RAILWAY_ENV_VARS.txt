REQUIRED ENVIRONMENT VARIABLES FOR RAILWAY:

1. DATABASE_URL
   - Your Supabase connection string
   - Format: postgresql://postgres.xxx:password@aws-0-region.pooler.supabase.com:6543/postgres

2. NEXT_PUBLIC_API_BASE_URL
   - Set to: http://localhost:5000
   - (Backend runs on same container at port 5000)

3. NEXTAUTH_URL
   - Set to: https://your-app.railway.app
   - (Your Railway domain)

4. NEXTAUTH_SECRET
   - Generate with: openssl rand -base64 32
   - Or any random 32+ character string

5. JWT_SECRET_KEY
   - Generate with: openssl rand -base64 32
   - Or any random 32+ character string

OPTIONAL (if you have them configured):
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- GITHUB_CLIENT_ID
- GITHUB_CLIENT_SECRET
- SMTP credentials for emails

HOW TO SET IN RAILWAY:
1. Go to your Railway project
2. Click on your service
3. Go to "Variables" tab
4. Click "New Variable"
5. Add each variable above

After setting variables, redeploy!
