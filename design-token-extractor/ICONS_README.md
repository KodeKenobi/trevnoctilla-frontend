# Icon Setup

You need to create icon files for the extension. Create these sizes:

- `icons/icon16.png` - 16x16 pixels
- `icons/icon32.png` - 32x32 pixels
- `icons/icon48.png` - 48x48 pixels
- `icons/icon128.png` - 128x128 pixels

## Quick Icon Creation

You can:

1. Use a design tool (Figma, Photoshop, etc.) to create a simple icon
2. Use an online icon generator
3. Use a placeholder image for development

### Simple Icon Ideas:

- A color palette/swatch icon
- A design token symbol
- A paintbrush with tokens
- Abstract geometric shapes representing design elements

The icon should represent design tokens, colors, or design systems.
